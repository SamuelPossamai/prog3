	######  DIRETRIZES PARA O BOM FUNCIONAMENTO DO JOGO ######

- Para executar o jogo,  abra o terminal no diretorio "Napoleao/Interface" e execute o comando ./Interface;
- Para compilar o programa, abra o terminal no diretorio "Napoleao/Interface" e execute o comando "make". Esse comando vai gerar diversos outros arquivos, para deixar apenas os que importam, de o comando "make clean" no mesmo diretorio;
- O jogo tem um modo para mostrar as cartas do oponente. Para ativar, vá no diretorio "Napoleao", abra o arquivo "tela.h" e mude a constante "MOSTRA" para "true". Depois, faça o passo acima para recompilar o programa;
- O jogo tem um modo para simular os jogadores. Para ativar, vá no diretorio "Napoleao", abra o arquivo "tela.h" e mude a constante "SIMULACAO" para "true". Depois, faça o passo da linha 4 para recompilar o programa;
- Após o jogo iniciar, haverá algumas caixas de diálogo. Na caixa "Rodadas", insira apenas um numero inteiro positivo maior que 0 e menor que 1000. Algo além disso poderá acarretar em erros e em uma execução imprevisível;
- Logo após o fim das caixas de diálogo iniciais, clique no botão "Iniciar" para efetivamente iniciar o jogo;
- Siga as instruções do jogo, que estão na caixa "Informações". Fique atento à essa caixa, pois ela mostrará o andamento do jogo;
- Para selecionar o numero de turnos e o naipe de trunfo, clique em um dos botões que irão aparacer. Para selecionar uma carta, apenas clique nela. Se mostrar a parte de trás da carta com um 'X' vermelho, quer dizer que não poderá clicar nela, se clicar não irá acontecer nada;
- Se você clicar na carta, e ela não aparecer no monte central, ou se quando for a vez do adversário a carta não aparecer, quer dizer que a carta que ele jogou é menor que a carta que já estava no monte;
- As regras do jogo estarão descritas em um arquivo .pdf no diretorio "Napoleao";
- Para qualquer dúvida adicional, ou para relatar algum bug, contate o criador do jogo (email: vitor.russi@hotmail.com).
