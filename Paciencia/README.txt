######  README ######

- Para executar o jogo,  abra o terminal no diretorio "Interface" e execute o comando ./Interface;

- Para compilar o programa, abra o terminal no diretorio "Paciencia/Interface" e execute o comando "make". Esse comando vai gerar diversos outros arquivos, para deixar apenas os que importam, de o comando "make clean" no mesmo diretorio;

- Após o jogo iniciar, haverá uma caixa de diálogo. Para entrada do nome do jogador;

- Para realizar uma jogada clique na carta que você quer mover, e logo em seguida para onde deseja ir, retornando uma mensagem de jogada inválida, se a jogada não for permitida.
