Selecionando Projeto

    ~$ qmake [Nome do Projeto]
    
    ex:
    
    ~$ qmake BlackJack

Compilando

    ~$ make

Executando

    ~$ make run
    
Limpando

    ~$ make veryclean
    
    antes de evniar projeto para o repositorio:
    
    ~$ make extraclean
